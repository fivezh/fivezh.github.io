html解析

beautifulsoup对html解析相比htmlparse更智能便捷。

  • 通过css选择器定位元素：soup.select(‘.classname > li’)选择.classname累下的li节点
  • beautifulsoup中Tag节点逐级选择，注意在元素有子节点时string无法获取内容，需要通过逐级定位元素后才能通过string获取内容
  • contents获取所有子节点，排查位置时比较便捷
